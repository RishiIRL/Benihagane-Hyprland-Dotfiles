# Display Modes

| Keybind | Action |
|---------|--------|
| SUPER + F1 | No Distractions Mode |
| SUPER + F2 | Opaque Mode |

## No Distractions Mode

- Removes gaps
- Removes borders
- Disables animations
- Disables blur
- Disables shadows
- Removes rounded corners
- Makes all windows fully opaque

Press the shortcut again to restore the normal configuration.

---

## Opaque Mode

Makes every window fully opaque.

Press the shortcut again to restore the normal configuration.
